# Database Schema

The primary operational data store is SQLite. The schema is managed via SQLAlchemy ORM and Alembic migrations.

## Table: `leads`

Stores inbound leads and their corresponding AI analysis results.

| Column | Type | Constraints | Description |
|---|---|---|---|
| `id` | VARCHAR(36) | PRIMARY KEY | Unique UUID for the lead record. |
| `full_name` | VARCHAR | NOT NULL | The lead's full name. |
| `email` | VARCHAR | NOT NULL | The lead's email address. Indexed for faster lookups. |
| `company_name` | VARCHAR | NULL | The lead's company name. |
| `job_title` | VARCHAR | NULL | The lead's job title. |
| `phone` | VARCHAR | NULL | The lead's phone number. |
| `company_size` | VARCHAR | NULL | Estimated size of the company. |
| `budget_range` | VARCHAR | NULL | Estimated budget. |
| `message` | TEXT | NOT NULL | The original message submitted by the lead. |
| `status` | VARCHAR | NOT NULL | Current status (e.g., `qualified`, `low_context`, `filtered`, `qualified_fallback`). |
| `lead_score` | INTEGER | NULL | AI-generated score (0-100). |
| `priority_tier` | VARCHAR | NULL | Priority tier categorization (e.g., `High`, `Cold`). |
| `intent_summary` | TEXT | NULL | Summary of the user's intent generated by AI. |
| `suggested_opener` | TEXT | NULL | AI-generated email response opener. |
| `red_flags` | JSON | NULL | JSON array of detected red flags or error states. |
| `received_at` | DATETIME | NOT NULL | Timestamp when the webhook was received (UTC). Default: `now()`. |
| `processed_at` | DATETIME | NULL | Timestamp when processing finished (UTC). |

### Schema Decisions
- **UUIDs** are used for primary keys to prevent exposing total lead counts and easily support distributed inserts if the database scales.
- **JSON for Red Flags** allows storing variable-length arrays natively in the database without requiring a separate table.
- **Nullable Analysis Fields**: Fields like `lead_score` and `intent_summary` are nullable in the schema to support leads that are rejected early (e.g., `low_context`) without being sent to the AI.
